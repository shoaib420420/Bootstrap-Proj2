# Bootstrap-Proj2

## Description
**Bootstrap-Proj2** is a responsive landing page designed for a digital agency or portfolio. It showcases services, statistical achievements, latest news/blogs, and contact information. The project utilizes a modern and vibrant design with a focus on user experience and responsiveness across all devices.

## Features
- **Responsive Navbar**: A collapsible navigation bar that includes a search functionality, adapting seamlessly to mobile and desktop screens.
- **Hero Section**: An engaging banner area with a call-to-action (CTA) to encourage user interaction.
- **Services Section**: Highlighted services presented with descriptive text and accompanying imagery.
- **Counter Section**: A dynamic section displaying statistical achievements (e.g., Happy Clients, Happy Developers) with vector icons.
- **Blog Section**: A grid layout displaying the latest news and articles with images and "Read More" links.
- **Footer**: A comprehensive footer containing "About," "Services," and "Social Contacts" sections, along with a subscription form and social media links.
- **Responsive Design**: The layout is fully adaptable, ensuring a consistent and professional look on Desktops, Tablets, and Mobile devices.

## Technologies Used
- **HTML5**: For semantic structure and content.
- **CSS3**: For custom styling, layout control (Flexbox), and media queries.
- **Bootstrap 4**: A powerful front-end framework used for the responsive grid system and pre-built components.
- **Bootstrap Icons**: For high-quality vector icons used throughout the site.
- **jQuery & Popper.js**: Essential libraries for enabling Bootstrap's interactive components.

## Installation & Usage
To run this project locally on your machine:

1.  **Clone the repository** (if applicable) or download the project files.
2.  **Navigate to the project directory** where you have saved the files.
3.  **Open `index.html`** directly in any modern web browser (Google Chrome, Firefox, Edge, Safari).

### Folder Structure
- `root/`: Contains the main `index.html` file.
- `css/`: Contains the `style.css` file for custom styling.
- `images/`: Stores all image assets used in the project (logos, blog images, banners, etc.).
